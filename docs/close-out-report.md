## 五、仍需未来优化的项

1. **公网 URL 部署**：需用户提供云平台凭据（Vercel/阿里云函数）
2. **虚拟滚动**：长列表 100+ 场景下 DOM 节点累积（Lighthouse TBT 优化）
3. **真实用户指标**：所有依赖真实用户行为的指标（CTR/完读率/举报漏判率）暂无法出具
4. **安全准确率提升**：pornography Recall 0.850 仍有提升空间（边界模糊样本如"春药"被判 allow）
5. **LLM 诊断升级**：数据回流诊断当前为硬编码阈值，DATA_DIAGNOSIS Prompt 留作未来 LLM 诊断
6. **热榜上榜通知**：真实埋点接入后触发 HOT_RANK/MILESTONE_VIEWS
7. **图片视觉 API**：素材合规校验当前为文本启发式，真视觉 API 待集成
